# Snapchat Memory Downloader

A Python tool to automate downloading, extracting, and fixing metadata for Snapchat Memories exports.

After researching for hours, I couldn't find any proper tool which cloud Download files from snapchat, Restores correct creation date, location (GPS), and timezone metadata, to ensuring your photos and videos appear with accurate time and place when added to iCloud or IOS devices in general.
So, I created this script for myself and it helped me saved me so much time and tedeous tasks. 

This is super easy to setup, hope this help you too! 

---

## Features

- Download media files (including ZIP archives) from Snapchat Memories JSON export
- Extract photos and videos from ZIP files automatically
- Add/repair EXIF metadata:
  - Correct capture date and time (including proper timezone handling)
  - GPS coordinates, so your media locations show up on maps
  - Timezone offset tags (crucial for accurate display in iCloud/iPhone)
- Cleans up extracted folders and ZIP files after processing
- Works on both Windows and macOS/Linux (requires Python 3.9+ and [ExifTool](https://exiftool.org/))

---

## Prerequisites
- VScode (Reccomended)
- Python 3.9+
- [ExifTool](https://exiftool.org/) must be installed and available in your PATH
- `requests` Python package (`pip install requests`)
- `sudo apt update` & `sudo apt install libimage-exiftool-perl`
- If your data is from other locations, update the timezone logic in `set_metadata()` accordingly
